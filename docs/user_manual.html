01

시작하기 — 회원가입 및 로그인

회원가입

1
회원가입 페이지 접속로그인 화면 하단의 '회원가입' 링크를 클릭합니다.
2
필수 정보 입력아이디, 이름, 이메일, 조직명, 직책, 비밀번호를 모두 입력합니다.
3
승인 대기가입 완료 후 관리자 승인을 기다립니다. 첫 번째 가입자는 자동으로 관리자 권한이 부여됩니다.
4
로그인승인 완료 후 아이디와 비밀번호로 로그인합니다.
아이디와 비밀번호에는 한글을 사용할 수 없습니다. 영문, 숫자, 특수문자만 입력 가능합니다.

로그인

항목설명
아이디가입 시 등록한 아이디 (영문/숫자)
비밀번호가입 시 설정한 비밀번호 (최소 4자)
로그인 상태 유지체크 시 브라우저를 닫아도 30일간 로그인 유지

비밀번호 찾기

1
비밀번호를 잊으셨나요? 클릭로그인 화면 하단 링크를 클릭합니다.
2
가입 이메일 입력가입 시 등록한 이메일 주소를 입력하고 '재설정 링크 발송'을 클릭합니다.
3
이메일 확인받은 편지함에서 재설정 링크를 클릭합니다. 링크는 1시간 동안 유효합니다.
4
새 비밀번호 설정새 비밀번호를 입력하고 저장합니다.
02

대시보드 — 프로젝트 관리

대시보드 구성

로그인하면 대시보드가 표시됩니다. 내 프로젝트 목록과 다른 사용자가 공개한 프로젝트를 조회할 수 있습니다.

프로젝트 생성

1
새 프로젝트 클릭좌측 사이드바 하단의 '＋ 새 프로젝트'를 클릭합니다.
2
프로젝트 정보 입력프로젝트명(필수), 설명, 조직명, 사업 담당자, 수행 담당자, 공개 여부를 입력합니다.
3
생성 완료프로젝트 상세 페이지로 이동하며 6단계 분석을 시작할 수 있습니다.

프로젝트 공개/비공개 설정

프로젝트 상세 화면에서 공개 여부를 토글할 수 있습니다. 공개로 설정하면 다른 사용자도 조회할 수 있습니다 (편집은 소유자만 가능).

좌측 사이드바에서 내 모든 프로젝트를 조직별로 확인하고 빠르게 이동할 수 있습니다.
03

6단계 사업모델 분석 모듈

각 프로젝트는 아래 6단계 모듈로 구성됩니다. 순서대로 작성하는 것을 권장하지만, 언제든지 원하는 단계부터 시작할 수 있습니다.

Step 01

🎯 사업 정의

사업의 기본 틀을 정의합니다.

  • 사업 도메인
  • 핵심 제품·서비스 범위
  • 매출 구조 및 주요 고객군
  • 기업 규모 및 보유 역량
  • 해결하려는 고객 문제
Step 02

🌐 환경 분석

외부 환경을 5가지 프레임워크로 분석합니다.

  • PESTEL 분석
  • 5 Forces 분석
  • SWOT 분석
  • VRIO 분석
  • 고객 세그먼트 맵
Step 03

💡 가치 설계

제품과 서비스의 가치를 설계합니다.

  • 핵심 제품/서비스
  • 차별화 포인트
  • 고객 경험 여정
  • 핵심 자원·활동·파트너십
Step 04

💰 수익 구조

수익 모델과 비용 구조를 정의합니다.

  • 수익 모델 유형
  • 가격 전략
  • 고객 획득 채널
  • 비용 구조 / 단위 경제성
Step 05

🚀 실행 체계

실행 계획을 수립합니다.

  • 실행 로드맵
  • 핵심 성과지표 (KPI)
  • 조직 및 역할
  • 필요 자원·예산 / 리스크
Step 06

🔬 검증·정제

가설을 검증하고 모델을 개선합니다.

  • 검증 가설 (핵심 가정)
  • 검증 방법 (MVP/실험)
  • 수집된 데이터/피드백
  • 학습·인사이트 / 피벗

데이터 입력 및 저장

1
모듈 탭 클릭프로젝트 상세 화면에서 원하는 단계 탭을 클릭합니다.
2
입력 필드 작성각 항목을 입력합니다. 최대 2,000자까지 입력 가능하며 우측 하단에 글자 수가 표시됩니다.
3
💾 저장 클릭작성 완료 후 저장 버튼을 클릭합니다. 저장하지 않으면 데이터가 유실됩니다.
입력 데이터는 저장 후에만 AI 분석에 사용됩니다. 반드시 저장 후 AI 분석을 요청하세요.
04

AI 분석 기능 활용

Claude AI(BCG 수석 컨설턴트 수준)를 활용한 다양한 분석 기능을 제공합니다.

AI 분석 종류

기능버튼설명
AI 자동 데이터 입력✦ AI 자동 데이터 입력사업 개요를 입력하면 AI가 모든 필드를 자동으로 작성합니다.
AI 분석 실행✦ AI 분석입력된 데이터를 기반으로 전문 컨설팅 수준의 분석 보고서를 생성합니다.
필드별 AI 질문각 필드 우측 ? 아이콘특정 필드에 대해 AI에게 질문하고 답변을 입력란에 반영합니다.

AI 자동 데이터 입력 사용법

1
AI 자동 데이터 입력 버튼 클릭모듈 상단의 '✦ AI 자동 데이터 입력' 버튼을 클릭합니다.
2
사업 개요 입력팝업창에 사업의 핵심 내용을 간략히 입력합니다. (예: "B2B SaaS 기반 HR 관리 솔루션, 중소기업 대상")
3
AI 입력 확인AI가 생성한 내용이 각 필드에 자동으로 채워집니다. 내용을 검토하고 필요 시 수정 후 저장합니다.

AI 분석 실행 사용법

1
데이터 저장 확인입력 데이터가 저장되어 있어야 합니다.
2
✦ AI 분석 버튼 클릭분석 요청 버튼을 클릭합니다. 분석에는 30초~2분 정도 소요됩니다.
3
결과 확인화면 하단에 분석 보고서가 표시됩니다. 표, 게이지, 매트릭스 등 시각화 요소가 포함됩니다.

환경 분석(Step 02) 특이사항

환경 분석은 PESTEL, 5 Forces, SWOT, VRIO, 고객세그먼트 5개 서브모듈로 구성됩니다. 각 서브모듈을 탭으로 이동하며 개별 분석을 실행할 수 있습니다.

환경 분석은 서브모듈별로 독립적으로 저장·분석됩니다. 하나의 서브모듈을 저장해도 다른 서브모듈에는 영향이 없습니다.
05

파일 저장 및 다운로드

작성한 데이터와 AI 분석 결과를 PDF로 다운로드할 수 있습니다.

파일 저장 종류

종류버튼 위치포함 내용파일명 형식
단계별 파일 저장각 모듈 하단 '📄 단계 파일 저장'해당 단계 입력 데이터 + AI 분석 결과{프로젝트명}_Step{번호}_{단계명}_{월일시}.pdf
서브모듈 파일 저장환경분석 각 탭 '📄 파일 저장'해당 서브모듈 입력 데이터 + AI 분석 결과{프로젝트명}_Step02_{분석명}_{월일시}.pdf
전체 통합 보고서프로젝트 상단 '📋 전체 파일 저장'6단계 전체 입력 데이터 + AI 분석 결과{프로젝트명}_비즈니스모델설계_통합보고서_{월일시}.pdf
파일명 끝의 숫자(예: 051514)는 월일시(05월 15일 14시) 형식입니다. 여러 버전을 다운로드해도 파일명이 겹치지 않습니다.
AI 분석을 실행하지 않은 단계는 '입력 데이터'만 PDF에 포함됩니다. AI 분석 결과가 있는 단계는 입력 데이터와 분석 결과가 모두 포함됩니다.
06

질의응답(Q&A)

사업 전략에 관한 질문을 등록하고 AI로부터 전문적인 답변을 받을 수 있습니다.

새 질문 작성

1
질의응답 메뉴 클릭좌측 사이드바의 '💬 질의응답'을 클릭합니다.
2
새 질문 작성제목과 질문 내용을 입력합니다.
3
관련 프로젝트 선택 (선택)질문과 관련된 프로젝트를 선택하면 AI가 프로젝트 정보를 참고하여 더 정확한 답변을 제공합니다.
4
저장+AI답변 클릭AI 답변을 즉시 요청하거나 '저장만' 선택 후 나중에 요청할 수 있습니다.

답글 및 추가 질문

질문 상세 화면에서 답글을 추가할 수 있습니다. 답글에도 AI 답변을 요청할 수 있으며, AI는 이전 대화 맥락을 참고하여 답변합니다.

AI 답변은 백그라운드에서 생성됩니다. 답변 생성 중에는 로딩 표시가 나타나며, 완료되면 자동으로 화면에 표시됩니다.
07

검색

좌측 사이드바 상단의 검색창에서 프로젝트명, 입력 데이터, AI 분석 결과를 통합 검색할 수 있습니다.

검색 대상설명
프로젝트명/설명프로젝트 이름이나 설명에 검색어가 포함된 경우
모듈 입력 데이터각 단계에 입력한 텍스트 내용
AI 분석 결과AI가 생성한 분석 보고서 내용
검색 결과는 연관도 순으로 정렬되며, 검색어 주변 내용이 발췌문으로 표시됩니다.
08

계정 관리

프로필 수정

좌측 사이드바 하단의 이름을 클릭하면 프로필 페이지로 이동합니다.

항목수정 가능 여부
이름, 이메일, 조직명, 직책✅ 수정 가능
비밀번호✅ 현재 비밀번호 확인 후 변경 가능
아이디❌ 변경 불가

로그아웃

사이드바 하단의 '로그아웃' 버튼을 클릭합니다.
